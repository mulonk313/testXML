# PHP Products Management

## Overview
This is a PHP application for managing products stored in an XML file. It allows users to view, add, and manage products.

## Requirements
- PHP 7.4 or higher
- A local server environment (e.g., XAMPP, WAMP, or MAMP)
- A web browser
